# Instructions

1. Fill the `title` compulsorily
2. Optionally, fill the `callout`
3. Optionally, fill the `button_url`
4. Optionally, fill the `url`
5. To publish, set `draft` to `false`

# Notes

- Keep everything short (especially think of users on mobile phones)
- The `button_url` may be relative; indeed it should be relative if you are linking to a page on WWC
- If you forget to fill the `button_text` it will say ‘Learn more’ by default
- If you forget to fill the title, the banner will not show even if set the `draft` is set to `false`
- Always leave `headless` set to `true`
